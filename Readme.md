# Sprite Sheet Slicer

## Snapshot
![image](Snapshot.gif)

## How to set an animation
- Set the start and end frame index and then using F5 to update. 
- Or click two frame in the right canvas directly.

## Export structure
- ~OutputDir/**SpriteName**/**AnimationName**/**AnimationName_Index.png**
